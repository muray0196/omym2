# Concepts

- [AppConfig](app-config.md) - In-memory settings used by usecases after TOML loading.
- [Library](library.md) - Stable music Library identity with a mutable runtime root.
- [Track](track.md) - Current managed state of one music file known to OMYM2.
- [PathPolicy](path-policy.md) - Pure canonical path generation for Library-root-relative destinations.
- [Plan](plan.md) - Reviewed scheduled set of PlanActions before execution.
- [PlanAction](plan-action.md) - One reviewed action inside a Plan.
- [Run](run.md) - Execution attempt for applying a Plan.
- [FileEvent](file-event.md) - Durable operation log entry for Library music file mutations.
